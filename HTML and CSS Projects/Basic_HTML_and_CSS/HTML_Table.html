HTML Tags

Name Tag Symbol Action
Paragraph <p> Starts a new paragraph
Title <title> Title on the page's tab
Body <body> Body of the document
Main <main> Most important content of the body
Anchor <a> Creates link to another page
Style <style> Used to define style information
Comment <!--[comment here]--> Add comments to HTML that won't show on final page
Table <table> Creates a table
Table Row <tr> Creates table row
Table Heading <th> Table Header - bold and centered by default
Table Data <td> Data in the cells
Unordered List <ul> List with no sequence - usually bullet pointed
Ordered List <ol> List with items in sequence
List Item <li> Item on a list
